# Delivery Cost Calculator

Проект содержит реализацию функции расчета стоимости доставки и набор юнит-тестов на JUnit 5.

## Функциональность

Учитываются:
- расстояние доставки;
- габариты груза;
- хрупкость груза;
- загруженность службы доставки;
- минимальная стоимость доставки 400 рублей.

## Запуск тестов

```bash
mvn clean test
