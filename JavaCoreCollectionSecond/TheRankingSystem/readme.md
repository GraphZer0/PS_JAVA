# Esports Ranking System

## Описание

Система ранжирования игроков киберспортивной лиги.
Позволяет добавлять игроков, обновлять рейтинг, получать топ-N и определять ранг игрока.

## Архитектурные решения

- Используется TreeMap<Integer, Set<Player>> с сортировкой по убыванию.
- Отдельная Map<Integer, Player> обеспечивает быстрый поиск по id.
- Бизнес-логика вынесена в RankingSystem.
- Используется интерфейс RankingManager для снижения связности.
- Реализованы пользовательские исключения.

## Принципы ООП

- Инкапсуляция
- Разделение ответственности
- Полиморфизм через интерфейс
- Обработка checked исключений

## Возможности расширения

- Добавление базы данных
- Поддержка сезонов
- Потокобезопасная версия
- REST API
