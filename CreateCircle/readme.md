Задача 1: Создание класса Circle
Что нужно сделать?

1. [x] Создайте класс Circle, который будет представлять круг.
2. [x] Добавьте поле radius (тип данных: double) — радиус круга.
3. [x] Реализуйте:
    1. [x] Конструктор, который принимает значение радиуса и инициализирует поле radius.
    2. [x] Геттер getRadius(), возвращающий значение радиуса.
    3. [x] Сеттер setRadius(double radius), позволяющий изменять значение радиуса.
    4. [x] Метод getArea(), который возвращает площадь круга (π*r^2). Для числа π используйте Math.PI.

Задача 2: Добавление проверки на поля radius
Что нужно сделать?

1. [x] Модифицируйте класс Circle, добавив проверку в сеттере setRadius(double radius):
    1. [x] Если переданное значение меньше или равно нулю, выбросьте исключение IllegalArgumentException с сообщением «Радиус должен быть больше нуля».
2. [x] Добавьте проверку в конструкторе: если переданный радиус меньше или равен нулю, выбрасывайте то же исключение.

Задача 3: Реализация тестов для проверки работы класса
Что нужно сделать?

1. Создайте новый класс TestCircle для тестирования класса Circle.
2. Реализуйте методы для тестирования:
     1. Тест конструктора: проверьте, что при создании объекта с допустимым радиусом поле radius правильно инициализируется, а при недопустимом значении выбрасывается исключение.
     2. Тест метода setRadius: проверьте, что метод корректно изменяет значение радиуса при допустимом значении и выбрасывает исключение при недопустимом.
     3. Тест метода getArea: проверьте, что метод возвращает правильную площадь круга при заданном радиус
