Задача 1: Площадь прямоугольника 
Напишите программу, которая:

Запрашивает у пользователя длину и ширину прямоугольника.
Вычисляет площадь прямоугольника.
Выводит результат на экран. 
Используйте переменные типа double для хранения длины и ширины.

 
#Пример работы программы:


Введите длину прямоугольника: 5
Введите ширину прямоугольника: 3
Площадь прямоугольника: 15.0
