Запуск проекта

1. Убедитесь, что база данных H2 (students_db.mv.db) содержит импортированные данные из предыдущего задания.
2. Соберите проект:

mvn clean compile

3. Запустите приложение:

mvn exec:java -Dexec.mainClass="org.example.Main"

После запуска программа выполняет запросы через слой DAO и выводит результаты в консоль.

Реализованные методы DAO

* Получение всех студентов.
* Получение студентов, прошедших подготовительный курс.
* Получение студентов с баллом по математике выше заданного значения.
* Получение студентов по полу.
* Получение студентов по этнической группе.
* Получение студентов по комбинации пола и этнической группы.

Используемые технологии

* Java 17
* JDBC
* H2 Database
* Maven

Для защиты от SQL-инъекций все запросы с параметрами выполняются через PreparedStatement.
