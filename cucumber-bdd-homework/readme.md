# Cucumber BDD Homework

## Описание

Автоматизация тестирования функции авторизации пользователя с использованием:

- Java
- Maven
- Cucumber
- JUnit 5

## Что реализовано

- Background
- Scenario Outline
- Examples
- Hooks
- Tags
- Assertions

## Запуск

```bash
mvn clean test
```

## Отчет

После запуска тестов отчет находится:

```text
target/cucumber-report.html
```
