Library CLI App

Консольное приложение для управления библиотекой, разработанное на Java с использованием принципов объектно-ориентированного программирования и SOLID.

Функциональность

Приложение позволяет:

* Добавлять книги в библиотеку
* Просматривать список книг
* Добавлять читателей
* Просматривать список читателей
* Выдавать книги читателям
* Возвращать книги в библиотеку
* Просматривать список выдач

Используемые сущности

Book

Описывает книгу:

* id
* title
* author
* available

Reader

Описывает читателя:

* id
* name
* email

Loan

Описывает выдачу книги:

* id
* bookId
* readerId
* loanDate
* active

Применение принципов SOLID

SRP (Single Responsibility Principle)

Каждый класс отвечает только за одну задачу:

* Model — хранение данных
* DAO — работа с хранилищем
* Service — бизнес-логика
* ConsoleMenu — взаимодействие с пользователем

OCP (Open/Closed Principle)

Используются интерфейсы DAO, что позволяет добавлять новые реализации без изменения бизнес-логики.

DIP (Dependency Inversion Principle)

Сервисы зависят от абстракций (DAO-интерфейсов), а не от конкретных реализаций.
