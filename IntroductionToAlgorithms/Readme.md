Sorting Algorithms Project

Описание проекта

Данный проект реализует несколько алгоритмов сортировки массивов целых чисел на Java. Пользователь вводит размер массива, после чего генерируется случайный массив чисел. Программа выполняет сортировку с использованием:
	•	QuickSort (ручная реализация)
	•	MergeSort (ручная реализация)
	•	Стандартного метода Arrays.sort() из Java

Программа выводит:
	1.	Исходный массив
	2.	Массив, отсортированный каждым алгоритмом
	3.	Время выполнения каждой сортировки

Выводы
	1.	Все алгоритмы успешно сортируют массив.
	2.	Arrays.sort() показал наименьшее время работы.
	3.	MergeSort работает стабильно и предсказуемо.
	4.	QuickSort ручной реализации чуть медленнее, но отлично подходит для изучения принципов работы алгоритма.

Рекомендации:
	•	Для практических задач использовать Arrays.sort() — быстро и надежно.
	•	Для учебных и исследовательских целей использовать QuickSort и MergeSort, чтобы понимать внутреннюю работу алгоритмов.
