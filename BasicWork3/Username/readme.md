Задача 3. Создание программы для форматирования и валидации имен пользователей
        Создайте программу для форматирования и валидации имен пользователей. Программа должна обрабатывать введенное имя пользователя согласно следующим правилам:
        
        1. Имя пользователя должно начинаться с буквы.
        2. Может содержать буквы (a-z, A-Z), цифры (0-9) и знак подчеркивания (_).
        3. Длина имени должна быть от 3 до 20 символов.
        4. Все буквы должны быть преобразованы в нижний регистр.
        5. Несколько подряд идущих знаков подчеркивания должны быть заменены на один.
        Ваша программа должна:
        
        1. Запрашивать у пользователя ввод имени.
        2. Проверять валидность имени с помощью регулярного выражения.
        3. Если имя валидно, форматировать его согласно правилам.
        4. Выводить результат (отформатированное имя или сообщение об ошибке).
        5. Для решения задачи используйте классы Pattern, Matcher, методы matches(), replaceAll() и toLowerCase().
        
        
        Пример работы программы:
        Ввод: "User__NAME123"
        Вывод: "user_name123"
        Ввод: "123user"
        Вывод: "Ошибка: имя пользователя должно начинаться с буквы"
        
        Ввод: "a"
        Вывод: "Ошибка: имя пользователя должно содержать от 3 до 20 символов"
