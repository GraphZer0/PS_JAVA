Задача 2. Создание функции для удаления всех букв и пробелов из текста 
    Разработайте Java-программу, которая удаляет все буквы и пробелы из введенного пользователем текста, оставляя только цифры и знаки пунктуации. Для решения задачи используйте регулярное выражение и метод replaceAll() класса String.
    Ваша программа должна:
    
    1. Запрашивать у пользователя ввод текста.
    2. Применять регулярное выражение для удаления букв и пробелов.
    3. Выводить обработанный текст.
    
    Пример работы программы:
    Ввод: "Мой номер телефона: +7 (999) 123-45-67, а почтовый индекс: 123456."
    Вывод: "+7(999)123-45-67,123456."
