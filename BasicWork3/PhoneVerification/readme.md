Задача 1. Написание регулярного выражения для проверки телефона в международном формате
        Напишите Java-программу, которая проверяет, соответствует ли введенный пользователем номер телефона международному формату.
        Международный формат телефона должен соответствовать следующим правилам:

        1. начинается с символа «+» и кода страны (1-3 цифры);
        2. далее следует код города/оператора (1-3 цифры);
        3. затем идет основной номер (5-8 цифр);
        4. разделители между группами цифр могут быть пробелами, дефисами или отсутствовать вовсе.
        5. Пример валидных номеров:
            
            +7 999 123-45-67
            
            +1-555-123-4567
            
            +441234567890
            
        
        Ваша программа должна:
        
        1. Запрашивать у пользователя ввод номера телефона.
        2. Проверять введенный номер с помощью регулярного выражения.
        3. Выводить результат проверки (валидный номер или нет).
